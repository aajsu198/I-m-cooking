Inicio

Está página está hecha con el fin de demostrar los avances y conocimientos adquiridos
durante el transcurso del segundo semestre sobre la asignatura de Desarrollo en Páginas Web
llevada a cabo en la Preparatoria 8 Bachillerato "Técnico en programación web."

En el cual se trabajo con el lenguaje de HTML.5 y CSS. en el editor de código fuente
"Visual Studio Code" (el cual este está desarrollado por Microsoft para Windows, Linox, macOS y web),
donde se utilizo una extensión de visual studio Code el cual es "live server"
que nos permite ver los cambios al instante de ser realizarlos.
La asignatura de Desarrollo en Páginas web fue dirigida por la docente
Matzil Alejandra Hinojosa Posada la cuál nos instruyó hasta el final de la mejor manera posible.
